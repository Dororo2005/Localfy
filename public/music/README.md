# Local audio files

Place your own MP3 files in this directory. The demo catalog in `src/data/songs.ts`
expects these filenames:

- `demo-midnight-drive.mp3`
- `demo-neon-weather.mp3`
- `demo-still-morning.mp3`
- `demo-paper-planes.mp3`
- `demo-sunday-vinyl.mp3`
- `demo-soft-focus.mp3`
- `demo-late-checkout.mp3`
- `demo-golden-hour.mp3`

You can rename the files or add more songs as long as `audioUrl` in the catalog matches.
No copyrighted audio is included with the project.
